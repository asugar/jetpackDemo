## leetcode原题：一只青蛙一次可以跳上1级台阶，也可以跳上2级台阶。求该青蛙跳上一个 10 级的台阶总共有多少种跳法。
f(10) = f(9) +f(8)
f(9) = f(8) +f(7)
f(8) = f(7) +f(6)
...
f(3)=f(2) + f(1)
f(2)=2
f(1)=1

## 给你一个整数数组 nums ，找到其中最长严格递增子序列的长度。
输入：nums = [10,9,2,5,3,7,101,18]
输出：4
解释：最长递增子序列是 [2,3,7,101]，因此长度为 4 。
